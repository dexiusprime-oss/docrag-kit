# Test Documentation Project

This is a test project for DocRAG Kit integration testing.

## Features

- **Feature 1**: Automatic documentation indexing
- **Feature 2**: Semantic search capabilities  
- **Feature 3**: MCP integration with Kiro AI

## Installation

```bash
pip install docrag-kit
docrag init
```

## Usage

After initialization, you can:
1. Index your documentation with `docrag index`
2. Search through docs using MCP tools in Kiro
3. Get instant answers about your project

## Configuration

The system uses `.docrag/config.yaml` for configuration.
